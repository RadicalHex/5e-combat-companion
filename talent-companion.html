Character

Quick Start Guide

Editing Stats

Click any stat value to edit it directly. For sections with a ✏ icon, click it to open a multi-field edit mode, then click Save.

HP

Use − / + to adjust current HP. "Set" jumps to any value. Click the ✏ icon to edit max HP and temp HP.

Spell Slots

Click a pip to use or restore a spell slot. All slots refill on Long Rest. Use the ✏ icon to set your spellcasting ability, Spell Save DC, and Spell Attack bonus.

Prepared Spells

Below the slot pips, use + to add a spell (enter name and level). Click a spell name to expand its notes — double-click to edit them. Double-click the spell name to rename it. Remove a spell with ✕. Your list persists across sessions and is fully flexible — add or remove spells freely.

Skills & Saves

Click the pip next to a skill or save to cycle proficiency: hollow → teal (proficient) → ring (expertise).

Status

Track active spells, HP Effects (Aid, Heroes' Feast, etc.), and conditions. Add them with the + buttons in the Status panel.

Inventory

Use + Item or + Magic to add items. Expand an item to see its description — double-click the description to edit it. Magic items can be toggled Equipped or Attuned.

Feats

Click ⚙ Manage Feats to browse the feat catalogue (SRD 5.1 + Psionic feats from The Talent). Check a feat to add it — its description pre-fills and is fully editable. Custom feats can be added from the modal footer. Expand a feat in the panel to edit its description or double-click the name to rename it.

Short Rest

Open the Rests menu and click Short Rest. Choose how many hit dice to spend, then click Roll to see individual results (each die + CON modifier). Click Heal to apply the total to your HP and spend the dice. Cancel at any point to do nothing.

Lore Book

Click 📖 Lore in the header to open the floating Lore Book panel. Use + to create named pages (NPCs, Locations, Factions, etc.). Click the page dropdown to switch between pages. Use ✎ to rename the active page and ✕ to delete it. Type directly in the panel to edit — content saves automatically. Use Export to download your lore as a JSON backup, and Import to restore it.

Stat Block Viewer

Drag a PDF or image onto the page to load a stat block. Drag the viewer panel to reposition it.

Saving

Everything saves automatically to this browser. Use Export / Import (in the Rests menu) to back up or transfer your character.
